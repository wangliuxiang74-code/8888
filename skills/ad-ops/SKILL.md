# SKILL.md - 跨境电商广告投流

## Description
Amazon PPC、Facebook Ads、TikTok Ads广告投放与优化。

## When to Use
- 新品推广
- 冲排名
- 清库存
- 品牌曝光

## Amazon PPC

### 广告类型
| 类型 | 用途 | 建议预算 |
|------|------|---------|
| SP自动 | 测词、拓流 | $20-50/天 |
| SP精准 | 推排名 | $30-100/天 |
| SP广泛 | 拓词 | $20-30/天 |
| SB品牌 | 品牌曝光 | $50+/天 |
| SD展示 | 再营销 | $30+/天 |

### 关键词策略
1. **新品期**：自动广告跑数据
2. **成长期**：筛选出单词转精准
3. **成熟期**：防御性投竞品ASIN

### ACoS控制
- 目标ACoS < 利润率
- 童装/玩具利润率30-40%，ACoS目标20-30%
- 亏本品控ACoS，推排名期可容忍短期亏损

### 优化动作
- 每周：加否定词、调竞价
- 每月：分析搜索词报告
- 每季：重新分配预算

## Facebook Ads

### 受众定位
- **兴趣**：童装→Parents、Baby clothing
- **玩具**：Toys、Educational toys、STEM
- **行为**：Online shoppers、Engaged shoppers
- **自定义**：网站访客、加购未付款
- **Lookalike**：1-3%相似受众

### 素材类型
- 单图：产品图+卖点
- 轮播：多SKU展示
- 视频：15s使用场景
- 动态：DPA再营销

### 预算分配
- 测试期：$20-50/天，5-10个素材
- 放量期： winners × 3-5倍预算
- 回收期：ROAS > 2.5维持，< 2关停

## TikTok Ads

### 特点
- 视频为主，图文效果差
- 创意>定向，素材决定成败
- 年轻妈妈群体活跃

### 素材方向
- 开箱视频
- 宝宝上身效果
- 对比测评
- "妈妈种草"真人出镜

### 投放策略
- Spark Ads：加热自然流量好的视频
- 商城广告：直接带货
- 预算：$30-100/天起步

## 通用原则

### 数据指标
| 指标 | 健康值 |
|------|--------|
| CTR | > 1% |
| CPC | 行业均值±20% |
| ROAS | > 2.5 |
| 转化率 | > 10% |

### 投放节奏
1. **Day 1-7**：测试素材/受众
2. **Day 8-14**：关停垃圾，放大赢家
3. **Day 15+**：稳定投放，持续优化

## Tools
- Helium 10 - Amazon广告
- Facebook Ads Manager
- TikTok Ads Manager
- 卖家精灵 - 竞品广告分析

## Notes
- 广告是放大器，产品不行别硬投
- 新品先养评论再开广告
- 童装玩具注意COPPA合规
- 节假日提前2周备素材
