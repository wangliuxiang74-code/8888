# SKILL.md - 内容数据分析

## Description
播放量分析、粉丝画像、内容效果、变现数据。

## When to Use
- 内容复盘
- 选题决策
- 粉丝分析
- 变现优化

## Workflow

### 1. 核心指标
**播放量：**
- 总播放量
- 完播率
- 平均观看时长

**互动：**
- 点赞率
- 评论率
- 转发率
- 收藏率

**粉丝：**
- 粉丝增长
- 粉丝画像（年龄/性别/地域）
- 活跃时间

### 2. 内容分析
**单集表现：**
- 播放量曲线
- 跳出点（哪里不看）
- 回看次数

**系列对比：**
- 哪类题材好
- 哪集是爆款
- 粉丝反馈关键词

**优化方向：**
- 完播率低→开头优化
- 点赞率低→情绪点不够
- 评论少→引导互动

### 3. 粉丝画像
**基础信息：**
- 性别比例
- 年龄分布
- 地域分布

**兴趣标签：**
- 还喜欢看什么
- 消费能力
- 活跃时段

**价值分层：**
- 核心粉（高频互动）
- 普通粉（偶尔互动）
- 路人（只看）

### 4. 变现数据
**广告：**
- 接单数量
- 单价
- 品牌类型

**带货：**
- GMV
- 佣金
- 转化率

**其他：**
- 打赏
- 付费内容
- 周边

### 5. 工具
- 抖音创作者中心
- B站创作中心
- 小红书创作中心
- 新榜/飞瓜/蝉妈妈

## Notes
- 数据是参考，别迷信
- 爆款有运气成分
- 持续优化比追求爆款重要
- 关注长期趋势
