# SKILL.md - 内容运营

## Description
多平台发布、算法优化、粉丝互动、内容规划。

## When to Use
- 发布内容
- 账号运营
- 流量优化
- 粉丝维护

## Workflow

### 1. 平台特性
**抖音：**
- 算法：完播率>点赞>评论>转发
- 黄金3秒：开头必须有钩子
- 更新：日更1-3集
- 互动：评论区引导追更

**B站：**
- 算法：播放量+三连（赞币藏）
- 标题党：但别标题党太假
- 封面：高清美图+标题
- 弹幕：引导发弹幕

**小红书：**
- 算法：互动率+收藏>点赞
- 封面：竖图、高颜值
- 标题：痛点/好奇/数字
- 标签：多带相关标签

### 2. 发布策略
**发布时间：**
- 早高峰：7-9点
- 午休：12-14点
- 晚高峰：18-22点
- 深夜：23-1点（情感向）

**发布节奏：**
- 新剧：日更3-5集，连更一周
- 老剧：周末加更/合集
- 节假日：特别篇

### 3. 内容规划
**选题库：**
- 甜宠向：霸总、先婚后爱、暗恋
- 虐恋向：替身、误会、复仇
- 玄幻向：修仙、系统、重生

**系列化：**
- 同一世界观多剧联动
- 角色客串
- 粉丝梗融入

### 4. 粉丝运营
**评论区：**
- 及时回复
- 置顶神评论
- 引导剧情讨论

**私域：**
- 粉丝群（剧透/福利）
- 周边预售
- 投票选剧情

**互动活动：**
- 最佳评论奖
- 二创大赛
- 角色人气投票

### 5. 数据监控
**核心指标：**
- 播放量、完播率
- 点赞、评论、转发
- 粉丝增长
- 单集转化率

**优化方向：**
- 完播率低→开头优化
- 点赞率低→情绪点不够
- 评论少→引导互动

## Tools
- 抖音创作者中心
- B站创作中心
- 小红书创作中心
- 新榜/飞瓜（数据分析）

## Notes
- 算法常变，多测试
- 粉丝是基本盘，别得罪
- 互动率比播放量重要
- 坚持更新，别断更
