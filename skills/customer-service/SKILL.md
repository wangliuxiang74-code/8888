# SKILL.md - 跨境电商客服话术

## Description
多语言客服自动回复、邮件模板、纠纷处理话术。

## When to Use
- 买家咨询产品
- 物流问题
- 退换货申请
- 差评挽回
- 纠纷仲裁

## Languages
- 英文（主）
- 西班牙语（美西市场）
- 葡萄牙语（巴西）
- 德语/法语（欧洲）
- 日语（日本站）

## 场景模板

### 1. 产品咨询
**English:**
```
Hi [Name],

Thanks for your interest in our [product]!

✅ Size: [details]
✅ Material: [details]  
✅ Age: [details]

Any other questions, feel free to ask!

Best,
[Store]
```

### 2. 物流查询
```
Hi [Name],

Your order [ORDER#] is on the way!

Tracking: [LINK]
Estimated delivery: [DATE]

Thanks for your patience!
```

### 3. 退货/换货
```
Sorry to hear that! We want to make it right.

Options:
1. Full refund - keep item
2. Replacement - free reship
3. Partial refund - partial keep

Which works for you?
```

### 4. 差评挽回
```
Hi [Name],

We noticed your feedback and sincerely apologize.

We'd love to make it right with:
- Full refund
- Free replacement
- [Discount] off next order

Please reply if we can help.
```

### 5. 纠纷应对
- 态度友好但坚定
- 提供证据（跟踪号、截图）
- 平台规则为依据
- 小额订单直接退款止损

## 童装/玩具特殊场景

### 尺码不合适
```
Sorry the size didn't fit!

For kids' clothes, we recommend:
- Measure child's height/chest
- Check our size chart
- Order one size up for growing room

We can offer [solution].
```

### 安全/材质质疑
```
Our products are:
✅ CPC certified (US)
✅ CE certified (EU)
✅ 100% [material] - A grade

Safety is our #1 priority for kids!
```

## Automation
- 设置自动回复（工作时间外）
- 关键词触发快捷回复
- 订单状态自动通知

## Notes
- 24h内必须回复（平台考核）
- 语气友好，多用表情
- 别承诺做不到的事
- 纠纷优先私了，别闹到平台仲裁
