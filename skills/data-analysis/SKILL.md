# SKILL.md - 数据分析

## Description
销售数据分析、广告ROI分析、市场趋势分析。

## When to Use
- 月度销售复盘
- 广告优化决策
- 选品数据支持
- 竞品监控
- 趋势预判

## Workflow

### 1. 销售数据分析
**核心指标：**
| 指标 | 计算方式 | 健康值 |
|------|---------|--------|
| 销售额 | 订单×客单价 | 月环比增长 |
| 订单量 | 总订单数 | 看趋势 |
| 客单价 | 销售额/订单量 | >$20 |
| 转化率 | 订单/访客 | >10% |
| 退货率 | 退货/总订单 | <5% |

**分析维度：**
- 按SKU：哪些是爆款/利润款/清货款
- 按时间：日/周/月趋势，季节性
- 按渠道：自然/广告/活动占比

### 2. 广告数据分析
**Amazon PPC：**
- ACoS（广告销售成本比）
- TACoS（总销售广告占比）
- ROAS（广告回报率）
- 点击率CTR、转化率CVR

**优化动作：**
- ACoS高的词→降竞价或否定
- CTR低的词→优化主图/标题
- CVR低的词→优化Listing/价格

**Facebook/TikTok：**
- ROAS目标 > 2.5
- CPA（单客获取成本）
- 受众分析（年龄/性别/地区）
- 素材效果排名

### 3. 竞品监控
**监控维度：**
- 价格变动（每天用工具抓）
- 排名变化（BSR追踪）
- Review增长（评价数/星级）
- 促销活动（LD/BD/Coupon）

**工具：**
- Helium 10 / Jungle Scout
- Keepa（价格历史）
- 卖家精灵

### 4. 市场趋势
**Google Trends：**
- 搜索热度趋势
- 地域分布
- 相关查询词

**站内数据：**
- 亚马逊品牌分析（ABA）
- 搜索词报告
- 品类增长趋势

### 5. 数据可视化
**日报/周报/月报模板：**
- 核心指标仪表盘
- 趋势折线图
- 占比饼图
- 异常预警标红

## Tools
- Excel（基础）
- Power BI / Tableau（高级）
- 卖家精灵数据导出
- Helium 10报表

## Notes
- 数据是决策依据，别凭感觉
- 异常数据先查原因再行动
- 定期复盘，形成习惯
- 数据滞后性，预判更重要
