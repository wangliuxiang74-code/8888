# SKILL.md - 财务核算

## Description
成本核算、利润分析、税务合规、资金管理。

## When to Use
- 单品利润测算
- 月度财务复盘
- 税务申报
- 资金规划
- 定价策略支持

## Workflow

### 1. 成本结构

**产品成本：**
- 工厂采购价
- 头程物流费
- 关税/增值税
- 仓储费

**平台成本：**
- 平台佣金（Amazon 15%/Shopee 5-6%/Temu核价）
- FBA配送费
- 广告费
- 促销折扣

**运营成本：**
- 人力成本
- 办公/软件费用
- 退货损耗
- 资金周转成本

### 2. 利润计算

**单SKU利润：**
```
毛利润 = 售价 - 产品成本 - 平台佣金 - FBA费 - 广告费
净利润 = 毛利润 - 运营成本分摊 - 退货损耗 - 税务
```

**利润率目标：**
- 毛利率 > 40%
- 净利率 > 20%

### 3. 定价策略

**成本加成法：**
```
售价 = 成本 / (1 - 目标利润率 - 平台费率)
```

**竞品对标法：**
- 比竞品低5-10%（打市场）
- 比竞品高（品牌溢价）
- 同价（红海竞争）

### 4. 税务合规

**美国站：**
- 销售税（Sales Tax）- 按州申报
- 所得税（年底申报）

**欧洲站：**
- VAT增值税（按月/季申报）
- EORI号（清关用）

**合规建议：**
- 找专业跨境会计
- 保留所有单据
- 按时申报，别偷税

### 5. 资金管理
- 回款周期（Amazon 14天/Shopee周结）
- 现金流预测
- 备货资金规划
- 外汇风险管理

## Tools
- QuickBooks / Xero
- 卖家精灵利润计算器
- Excel财务模型

## Notes
- 每笔订单都要算清楚利润
- 负利润产品及时止损
- 税务合规是底线
- 现金流比利润更重要
