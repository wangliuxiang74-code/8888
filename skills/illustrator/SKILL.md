# SKILL.md - 原画设计

## Description
角色设计、场景绘制、表情差分、服饰设计。

## When to Use
- 角色立绘
- 场景绘制
- 表情差分
- 服饰设计

## Workflow

### 1. 角色设计
**立绘：**
- 正面、侧面、背面三视图
- 身高比例（通常7-8头身）
- 标志性特征（发型/饰品）
- 配色方案（主色+辅色）

**表情差分：**
- 基础：平静
- 开心、难过、生气、害羞
- 惊讶、恐惧、嫌弃
- 每种至少5个

**服饰：**
- 日常装、礼服、睡衣
- 时代背景匹配
- 颜色区分角色性格

### 2. 场景设计
**室内：**
- 客厅、卧室、办公室
- 符合人物身份
- 细节体现性格

**室外：**
- 街道、公园、商场
- 氛围匹配剧情
- 远近层次

### 3. 绘画流程
1. 草图：确定构图
2. 线稿：清理线条
3. 底色：铺大色块
4. 细化：加阴影高光
5. 调整：整体统一

### 4. 工具
- Photoshop
- SAI / CSP
- Procreate（iPad）
- AI辅助：Midjourney/SD

### 5. 效率技巧
- 建立素材库（背景、道具）
- 分层管理
- 动作模板复用
- 批量导出

## Notes
- 风格统一，别一集一个样
- 主要角色精细，路人简化
- 版权注意，别抄其他作品
