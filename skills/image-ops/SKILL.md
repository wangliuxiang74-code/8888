# SKILL.md - 跨境电商图片处理

## Description
产品图片处理、主图优化、详情页设计、去水印、换背景。

## When to Use
- 做主图（白底/场景）
- 详情页排版
- 去水印/修图
- 尺寸图/对比图

## Workflow

### 1. 主图要求
Amazon/Shopee/Temu通用：
- 尺寸：2000×2000px
- 格式：JPG/PNG
- 第1张：白底图（RGB 255,255,255）
- 第2-5张：场景图+卖点图+尺寸图+细节图
- 第6张：对比图/视频封面

### 2. 图片处理工具
- **remove.bg** - 自动抠图换背景
- **Canva** - 详情页模板
- **Photopea** - 免费PS替代
- **Canvas** - AI生成场景图

### 3. 详情页结构
- 首屏：产品全貌+核心卖点
- 痛点场景：问题→解决方案
- 产品细节：材质/工艺/特写
- 使用场景：真人/宝宝上身
- 信任背书：认证/质检/评价
- 购买引导：尺码表/售后

### 4. 童装/玩具注意
- 真人上身图（安全打码）
- 材质细节（A类/纯棉）
- 年龄/身高尺码对照
- 安全认证标识

## AI Tools
- 生图：DALL-E / Midjourney / 即梦
- 扩图： outpainting
- 换脸：模特换脸保持统一
- 视频：可灵/Runway 产品动态展示

## Notes
- 图片决定点击率，别省功夫
- 每平台要求不同，上架前检查
- 侵权图片别用（品牌/卡通形象）
