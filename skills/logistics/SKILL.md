# SKILL.md - 物流管理

## Description
头程物流、FBA入仓、尾程配送、退货处理。

## When to Use
- 发货到海外
- FBA补货
- 物流商选择
- 退货处理
- 物流异常解决

## Workflow

### 1. 头程物流（中国→海外）
**方式对比：**
| 方式 | 时效 | 价格 | 适用 |
|------|------|------|------|
| 快递(DHL/FedEx) | 3-5天 | 贵 | 急件/高价值 |
| 空运 | 7-10天 | 中等 | 中小批量 |
| 海运 | 30-45天 | 便宜 | 大批量/非急 |
| 铁路 | 18-25天 | 中等 | 欧洲线 |

**物流商选择：**
- 传统货代（价格好，服务一般）
- 跨境专线（性价比高）
- 亚马逊AGL（省心但贵）

### 2. FBA入仓
**发货前准备：**
- 产品贴FNSKU标签
- 外箱贴亚马逊标签
- 称重测量（别超尺寸）
- 创建发货计划

**注意事项：**
- 混装/原厂包装别搞错
- 货件到达及时追踪
- 签收后检查数量差异

### 3. 尾程配送
**FBA配送：**
- 亚马逊负责，不用管
- 追踪买家收货

**FBM自发货：**
- 选择本地物流商
- 满足平台时效要求
- 提供追踪号

### 4. 退货处理
**退货类型：**
- 买家退货（亚马逊自动处理）
- FBA退货（退回仓库质检）
- 移除库存（弃置或移仓）

**退货分析：**
- 统计退货原因
- 质量问题反馈工厂改进
- 描述不符优化Listing

### 5. 物流成本优化
- 轻小件发空运
- 大件发海运
- 旺季提前2个月备货
- 多家物流商比价

## Tools
- 货代系统
- 17track物流追踪
- 亚马逊货件管理

## Notes
- 物流是成本大头，别省错地方
- 时效和成本平衡
- 旺季必爆仓，提前备货
- 物流异常及时沟通
