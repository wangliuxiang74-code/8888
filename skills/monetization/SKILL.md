# SKILL.md - 商务变现

## Description
广告接单、IP授权、电商带货、周边销售。

## When to Use
- 接广告
- 电商带货
- IP授权
- 周边开发

## Workflow

### 1. 广告变现
**类型：**
- 植入广告（剧情融入）
- 贴片广告（开头/结尾）
- 定制剧（品牌定制）

**报价：**
- 按粉丝量：X元/万粉
- 按播放量：CPM/CPC
- 按效果：CPA/CPS

**接单渠道：**
- 星图/蒲公英（官方）
- MCN机构
- 品牌直客

**注意：**
- 广告标识（#广告）
- 别接竞品
- 不影响剧情质量

### 2. 电商带货
**方式：**
- 视频挂车（小黄车）
- 直播带货
- 橱窗带货

**选品：**
- 和剧情相关（服饰/饰品）
- 高佣金
- 质量有保障

**平台：**
- 抖音：精选联盟
- 小红书：蒲公英

### 3. IP授权
**授权类型：**
- 角色形象授权
- 故事改编权
- 联名合作

**对象：**
- 游戏公司
- 影视公司
- 品牌方

**注意：**
- 版权登记
- 合同条款
- 授权范围

### 4. 周边销售
**类型：**
- 角色立牌/徽章
- 同款服饰
- 数字藏品

**渠道：**
- 淘宝/微店
- 平台小店
- 众筹（摩点）

### 5. 粉丝经济
**粉丝群：**
- 付费入群（提前看/福利）
- 打赏/充电

**内容付费：**
- 付费合集
- 会员专享

## Tools
- 星图/蒲公英
- 淘宝联盟
- 微店

## Notes
- 变现前提是内容好
- 别为赚钱牺牲质量
- 多元化收入来源
- 长期价值>短期利益
