## Amazon运营

### Listing要素
| 要素 | 要求 | 优化技巧 |
|------|------|---------|
| 标题 | ≤200字符 | 核心词+属性+卖点 |
| 五点 | 5条，每行≤500字符 | 痛点→方案→好处 |
| 描述 | A+页面 | 图文结合，品牌故事 |
| 后台关键词 | 250字节 | 不要重复标题词 |
| 图片 | 7张 | 白底+场景+细节 |
| 价格 | 有竞争力 | 参考竞品±10% |

### 标题公式
`品牌 + 核心关键词 + 属性（尺码/颜色/材质）+ 适用人群 + 卖点`

**童装示例：**
```
[Brand] Toddler Girls Dress Summer Cotton Casual Dresses for 2-6 Years Kids Short Sleeve Floral Print Playwear
```

### 五点写作
1. 核心卖点（材质/功能）
2. 适用场景（日常/派对/幼儿园）
3. 尺码建议（对照表+建议）
4. 洗护说明（耐用性）
5. 售后承诺（退换货政策）

### 关键指标
- **BSR排名**：越低越好
- **转化率**：>15%优秀，>10%及格
- **退货率**：<5%健康，<3%优秀
- **Review评分**：>4.3星

## Shopee运营

### 特点
- 东南亚市场（新马泰菲越印尼）
- 手机端为主，图片要竖版
- 免运活动流量大
- 聊聊回复率考核

### Listing优化
- 标题：60字符，关键词前置
- 图片：1:1或3:4竖版
- 视频：15-60s，提升转化
- 标签：10个热搜词

### 活动报名
- 免运活动（必须上）
- Flash Sale（报爆款）
- 金币返现（提升曝光）
- 直播（新兴流量入口）

## Temu运营

### 特点
- 全托管模式，只负责供货
- 核价严格，利润空间小
- 起量快，适合清库存
- 罚款多，注意质检

### 运营动作
1. 选品报款 → 买手核价
2. 备货入仓 → 质检上架
3. 报活动 → 冲销量
4. 补货/下架 → 管理动销

### 注意事项
- 图片白底，无水印
- 供货价要预留退货成本
- 首单少量测款，爆款加量
- 质检严格，次品率<1%

## 童装/玩具特殊注意

### 认证要求
- **美国**：CPC认证（儿童产品证书）
- **欧洲**：CE认证 + EN71
- **日本**：食品卫生法（接触皮肤）

### 标签要求
- 水洗标（材质成分）
- 尺码标
- 产地标
- 安全警示（小零件窒息风险）
