# SKILL.md - 分镜设计

## Description
分镜脚本、镜头设计、画面构图、节奏把控。

## When to Use
- 剧本转分镜
- 镜头设计
- 节奏规划
- 画面构图

## Workflow

### 1. 分镜要素
**景别：**
- 远景：环境展示
- 全景：人物全身
- 中景：膝盖以上
- 近景：胸部以上
- 特写：面部表情/细节

**角度：**
- 平视：正常视角
- 俯视：弱势/压抑
- 仰视：强势/压迫
- 斜视：不稳定/紧张

**运动：**
- 推：强调细节
- 拉：展示环境
- 摇：环境转移
- 移：跟随人物
- 跟：持续追踪

### 2. 分镜格式
```
镜号 | 景别 | 摄法 | 内容 | 台词/音效 | 时长
---|---|---|---|---|---
1 | 特写 | 推 | 女主眼泪滑落 | 心跳声 | 2s
2 | 近景 | 固定 | 男主回头看到女主 | 男主OS：她哭了？ | 3s
```

### 3. 节奏设计
**甜宠向：**
- 节奏轻快
- 特写多（表情）
- 慢镜头（心动瞬间）

**虐恋向：**
- 节奏慢，留白多
- 雨天/夜景
- 背影、空镜

**玄幻向：**
- 大场面远景
- 特效镜头
- 快节奏打斗

### 4. 转场技巧
- 直接切：剧情紧凑
- 淡入淡出：时间流逝
- 叠化：回忆/梦境
- 划像：场景切换

### 5. 参考学习
- 韩剧：分镜细腻
- 日漫：节奏把控
- 国产短剧：快节奏

## Tools
- Storyboarder（免费）
- Photoshop
- 手绘板

## Notes
- 每集至少有一个记忆点镜头
- 节约成本，别设计太难画的
- 和原画师沟通可行性
