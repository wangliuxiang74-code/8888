# SKILL.md - 供应链管理

## Description
工厂对接、采购管理、质检标准、库存控制。

## When to Use
- 找供应商
- 下采购单
- 质量检验
- 库存规划

## Workflow

### 1. 供应商管理
**筛选标准：**
- 产能（月产量）
- 质量（样品测试）
- 价格（多家比价）
- 交期（是否准时）
- 配合度（改款响应）

**评估维度：**
| 指标 | 权重 | 评分 |
|------|------|------|
| 质量 | 40% | 1-5分 |
| 价格 | 30% | 1-5分 |
| 交期 | 20% | 1-5分 |
| 服务 | 10% | 1-5分 |

**备份供应商：**
- 主力1-2家
- 备用2-3家
- 分散风险

### 2. 采购流程
**下单前：**
- 确认样品
- 谈好价格/交期
- 签订合同
- 支付定金（30%）

**生产中：**
- 中期检查（大客户）
- 进度跟踪
- 问题及时沟通

**发货前：**
- 全检/抽检
- 拍照确认
- 尾款支付

### 3. 质检标准
**童装：**
- 面料成分（纯棉/A类）
- 缝线密度
- 纽扣牢固度
- 尺码准确性
- 异味检查

**玩具：**
- 安全认证（CPC/CE）
- 小零件牢固（防窒息）
- 材质无毒
- 功能正常

**质检流程：**
- 来料检
- 过程检
- 成品检
- 出货检

### 4. 库存管理
**安全库存：**
- 爆款：30天库存
- 常规款：15天库存
- 测款：7天库存

**补货点：**
```
补货点 = 日均销量 × 采购周期 × 安全系数(1.5)
```

**库存周转：**
- 目标：月周转2-3次
- 滞销品：及时清货
- 季节性：提前2个月备货

### 5. 成本控制
**谈判技巧：**
- 量大价优
- 长期合作价
- 账期争取
- 次品率约定

**隐性成本：**
- 质检不良返工
- 交期延误损失
- 换供应商成本

## Tools
- 1688/阿里巴巴
- ERP系统
- Excel采购表

## Notes
- 供应商是合作伙伴，不是敌人
- 质量第一，价格第二
- 合同写明违约条款
- 定期评估，优胜劣汰
