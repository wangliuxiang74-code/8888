# SKILL.md - 视频剪辑

## Description
视频合成、转场特效、字幕包装、音效处理。

## When to Use
- 素材合成
- 添加特效
- 字幕包装
- 音效配乐

## Workflow

### 1. 剪辑流程
**导入素材：**
- 原画素材整理
- 配音文件对齐
- BGM准备

**粗剪：**
- 按分镜顺序拼接
- 调整时长
- 确定节奏

**精剪：**
- 添加转场
- 调整速度
- 画面特效

### 2. 特效处理
**转场：**
- 淡入淡出
- 叠化
- 划像
- 动态模糊

**画面特效：**
- 滤镜调色
- 光效
- 粒子效果
- 慢动作

### 3. 字幕包装
**字幕类型：**
- 对话字幕
- 内心OS
- 音效字（咚、啪）
- 强调字（放大/变色）

**风格：**
- 甜宠：粉色/爱心
- 虐恋：冷色调
- 玄幻：古风字体

### 4. 音效配乐
**BGM选择：**
- 甜宠：轻快钢琴
- 虐恋：悲伤弦乐
- 玄幻：史诗感

**音效：**
- 环境音
- 动作音
- 情绪音（心跳、叹息）

**音量平衡：**
- BGM -20dB
- 人声 0dB
- 音效视情况

### 5. 导出设置
**抖音：**
- 1080×1920（竖屏）
- 30fps
- H.264编码

**B站：**
- 1920×1080（横屏）
- 60fps（可选）
- 码率≥6Mbps

## Tools
- Premiere Pro
- After Effects（特效）
- 剪映（快速出片）
- Audition（音频）
