# SKILL.md - 配音导演

## Description
配音指导、音效设计、BGM搭配、混音处理。

## When to Use
- 配音录制
- 音效设计
- BGM选择
- 最终混音

## Workflow

### 1. 配音指导
**CV选择：**
- 音色匹配角色（甜美/御姐/少年/霸总）
- 情感表达能力
- 配合度（改音意愿）
- 价格（按集/按时长）

**录制指导：**
- 情绪铺垫（给CV讲戏）
- 语速控制（悲伤慢/紧张快）
- 重音强调（关键词）
- 呼吸声处理（自然/去除）

**台词优化：**
- 口语化调整
- 时长匹配画面
- 口型同步（如有）

### 2. 音效设计
**分类：**
- 环境音（街道/室内/自然）
- 动作音（走路/开门/打斗）
- 情绪音（心跳/叹息/紧张音效）
- UI音（提示音/转场音）

**来源：**
- 音效库（爱给网/ freesound）
- 自己录制
- AI生成

### 3. BGM搭配
**风格匹配：**
| 场景 | BGM风格 | 推荐 |
|------|---------|------|
| 甜宠 | 轻快钢琴 | 《菊次郎的夏天》风 |
| 虐恋 | 悲伤弦乐 | 《假如爱有天意》风 |
| 紧张 | 节奏感强 | 悬疑风 |
| 玄幻 | 史诗感 | 仙侠风 |

**音量控制：**
- BGM -20dB ~ -15dB
- 不压人声
- 情绪高点时BGM也高点

**版权问题：**
- 用免版权音乐
- 平台音乐库
- 原创BGM（长期考虑）

### 4. 混音处理
**音量平衡：**
- 人声：0dB（基准）
- BGM：-20dB
- 音效：视情况

**处理技巧：**
- 降噪（底噪去除）
- EQ均衡（人声突出）
- 压缩（音量统一）
- 混响（空间感）

**输出格式：**
- 采样率：44.1kHz/48kHz
- 位深：16bit/24bit
- 格式：WAV（无损）→转AAC（发布）

### 5. 工具
- Audition（专业）
- 剪映（快速）
- Reaper（免费）
- 音效库：爱给网、小森平

## Notes
- 配音是灵魂，别省钱
- CV情绪要给到位
- BGM别喧宾夺主
- 最后检查音画同步
