# 🏦 第一类：宏观因子 (20个)
**作用：判断黄金长期方向**

## 1. 实际利率 (Real Yield)
- **是什么**：你的存款利息 - 通胀 = 实际能拿到手的收益
- **怎么用**：实际利率是负的（比如-0.5%），你存钱反而亏钱，这时候黄金不要钱一样，大家都抢着买
- **信号**：负利率 → 黄金大涨 | 正利率 > 2% → 黄金承压

## 2. TIPS收益率
- **是什么**：美国通胀保护债券的收益率，本质上就是实际利率
- **怎么用**：TIPS收益率跌 = 实际利率跌 = 黄金涨
- **信号**：TIPS收益率 < 0 → 黄金利好

## 3. 美债10年期收益率
- **是什么**：美国政府借10年钱的利息
- **怎么用**：国债收益率下跌，说明大家不想要美元国债，要黄金
- **信号**：10年收益率从4%跌到3% → 黄金从4500涨到5000

## 4. 美债2年期收益率
- **是什么**：短期美国政府借2年钱的利息，反映美联储短期政策预期
- **怎么用**：2年期收益率暴跌 = 市场预期美联储要降息 = 黄金涨
- **信号**：2年期收益率单日大跌 → 黄金当日大涨

## 5. 利率预期 (Interest Rate Expectations)
- **是什么**：市场猜测未来美联储是加息还是降息
- **怎么用**：市场预期降息 = 美元弱 = 黄金强
- **信号**：市场预期从加息转向降息 → 黄金启动

## 6. 实际利率预期
- **是什么**：市场对未来实际利率的预期
- **怎么用**：预期实际利率下跌 = 黄金上涨
- **信号**：预期从1%降到-1% → 黄金涨10%

## 7. 负利率债券规模
- **是什么**：全球有多少债券是负利率的（即借钱还倒贴）
- **怎么用**：负利率债券越多，说明全球货币体系越烂，黄金越值钱
- **信号**：负利率债券从5万亿涨到10万亿 → 黄金从4500涨到6000

---

## 8. 美元指数DXY
- **是什么**：美元相对于欧元、日元、英磅等一篮子货币的强弱
- **怎么用**：黄金和美元是对头，美元涨黄金跌，美元跌黄金涨
- **信号**：DXY从105跌到100 → 黄金从4800涨到5300

## 9. DXY权重货币走势
- **是什么**：美元指数里欧元占57.6%，日元占13.6%
- **怎么用**：欧元涨1%对美元影响最大，欧元涨=美元跌=黄金涨
- **信号**：EUR/USD涨1% → 黄金涨0.5-1%

## 10. 美元兑G10货币
- **是什么**：美元对十个主要货币的整体强弱
- **怎么用**：比DXY更全面的美元强弱指标
- **信号**：美元对G10货币全线走弱 → 黄金大涨

## 11. 美元流动性
- **是什么**：美联储印了多少钱 + 逆回购市场里有多少钱
- **怎么用**：美联储资产负债表越大、逆回购越多 = 美元越不值钱 = 黄金越值钱
- **信号**：美联储资产负债表突破9万亿美元 → 黄金涨到5000+

## 12. 美元信用风险
- **是什么**：市场对美国政府借债能力的担忧
- **怎么用**：美国信用评级下调、债务上限危机 = 市场慌 = 黄金涨
- **信号**：美债评级下调 → 黄金单日涨2-3%

---

## 13. 全球央行净购金
- **是什么**：全世界央行每个月买多少黄金
- **怎么用**：央行是黄金最大单一买家，2022年起全球央行疯买
- **信号**：月净购金 > 50吨 → 黄金支撑强 | > 100吨 → 黄金要暴涨

## 14. 中国央行购金
- **是什么**：中国央行买黄金的速度
- **怎么用**：中国连续几个月增持 = 重大利好
- **信号**：中国连续3月增持 → 黄金涨5-10%

## 15. 俄罗斯央行购金
- **是什么**：俄罗斯被制裁后疯狂买黄金去美元化
- **怎么用**：俄罗斯买黄金 = 去美元化标志
- **信号**：俄罗斯持续购金 → 黄金长期利好

## 16. 央行售金
- **是什么**：哪些央行在卖黄金
- **怎么用**：央行卖 = 短期利空
- **信号**：IMF/欧洲央行售金 → 黄金短线承压

## 17. 央行黄金储备占比
- **是什么**：一个国家外汇储备里黄金占多少
- **怎么用**：新兴市场国家在降低美元占比、提高黄金占比
- **信号**：各国央行黄金储备占比提升 → 黄金长期利好

## 18. IMF售金
- **是什么**：国际货币基金组织卖黄金
- **怎么用**：IMF卖黄金 = 短期增加供给 = 黄金跌
- **信号**：IMF公告售金 → 黄金短线跌1-2%

---

## 19. CPI同比
- **是什么**：消费者物价指数，通胀率
- **怎么用**：通胀涨 = 黄金涨（黄金是通胀保值资产）
- **信号**：CPI从3%涨到5% → 黄金从4800涨到5200

## 20. PPI同比
- **是什么**：生产者物价指数，工厂端的通胀
- **怎么用**：PPI涨 = 未来CPI要涨 = 黄金涨
- **信号**：PPI超预期 → 黄金涨

## 21. 核心PCE
- **是什么**：美联储最看重的通胀指标
- **怎么用**：核心PCE涨 = 美联储要加息 = 黄金跌
- **信号**：核心PCE > 2% → 黄金承压 | < 2% → 黄金反弹

## 22. 通胀预期 (5年期BEI)
- **是什么**：市场对未来5年通胀的预期
- **怎么用**：通胀预期涨 = 黄金涨
- **信号**：BEI从2%涨到3% → 黄金涨5%

## 23. 盈亏平衡利率
- **是什么**：名义利率 - 通胀预期 = 实际利率
- **怎么用**：和实际利率一个意思
- **信号**：盈亏平衡利率跌 → 黄金涨

# 📊 第二类：机构订单因子 (15个)
**作用：跟着主力资金走**

## 24. COT大型投机商净头寸
- **是什么**：对冲基金/大户在期货市场有多少净多头或净空头
- **怎么用**：对冲基金是"追涨杀跌"的散户代表，他们持仓创历史新高 = 要见顶了
- **信号**：净多头 > 30万手 → 接近顶部 | 净空头 > 20万手 → 接近底部

## 25. COT商业套保商净头寸
- **是什么**：黄金生产商（卖黄金）和消费商（买黄金）的净头寸
- **怎么用**：生产商大量做空 = 他们在对冲风险 = 预计价格要跌 | 消费商大量做多 = 预计价格要涨
- **信号**：生产商净空头创历史 → 底部信号 | 消费商净多头创历史 → 顶部信号

## 26. COT持仓变化
- **是什么**：每周持仓的变化
- **怎么用**：连续3周同向变化 = 趋势形成
- **信号**：连续3周净多头增加 → 上涨趋势确立

## 27. COMEX黄金期货成交量
- **是什么**：纽约商品交易所黄金期货每天交易多少手
- **怎么用**：成交量放大 = 方向要出来了
- **信号**：成交量突然放大2倍 → 突破在即

## 28. COMEX黄金期货持仓量
- **是什么**：有多少手期货没平仓
- **怎么用**：持仓量历史新高 = 风险累积 = 随时爆仓
- **信号**：持仓量创新高 + 价格盘整 → 即将突破

## 29. 期货升贴水 (Contango/Backwardation)
- **是什么**：期货价格比现货贵还是便宜
- **怎么用**：期货溢价(Contango) = 机构看涨 | 期货贴水 = 机构看跌
- **信号**：Contango > 2% → 机构看涨黄金

## 30. 期货价差 (Basis)
- **是什么**：期货和现货的价差
- **怎么用**：价差突然放大 = 市场紧张
- **信号**：价差从5块扩大到50块 → 波动要放大

---

## 31. GLD净流入
- **是什么**：全球最大黄金ETF (SPDR Gold Trust) 每天净申购/赎回多少
- **怎么用**：单日净流入 > 5亿美元 = 机构大批买入
- **信号**：GLD单日净流入5亿+ → 黄金当日涨1-2%

## 32. GLD净流出
- **是什么**：GLD每天净赎回多少
- **怎么用**：单日净流出 > 5亿美元 = 机构大批跑路
- **信号**：GLD净流出5亿+ → 黄金当日跌1-2%

## 33. SLV净流入
- **是什么**：白银ETF的流向
- **怎么用**：金银一起涨，银更强 = 黄金还要涨
- **信号**：SLV净流入 > GLD净流入 → 银强金弱

## 34. GLD溢价/折价
- **是什么**：ETF交易价格 vs 黄金现货价格的差
- **怎么用**：溢价 > 1% = 需求旺盛，大家都想买 | 折价 = 供过于求
- **信号**：溢价从0.5%涨到2% → 黄金还要涨

## 35. ETF持仓集中度
- **是什么**：前10大持仓机构占了多少比例
- **怎么用**：集中度高 = 几个大户决定走势
- **信号**：前10大持仓 > 70% → 容易被控盘

---

## 36. 流动性耗尽 (Liquidity Sweep)
- **是什么**：主力先假突破高点/低点，扫掉散户止损，然后反着做
- **怎么用**：看到快速突破+放量+快速反转 = 主力在扫止损
- **信号**：突破后30分钟内反转 → 即将反方向开仓

## 37. 暗池订单分布
- **是什么**：CME期货80%交易在电子平台外（暗池），这些价位是真正支撑阻力
- **怎么用**：价格到暗池买单密集区 → 有支撑 | 到卖单密集区 → 有阻力
- **信号**：暗池大量买单在4900 → 4900是强支撑

## 38. 订单块 (Order Block)
- **是什么**：主力建仓后会留下"脚印"，连续几根K线同方向
- **怎么用**：价格回到订单块区域 = 获得支撑
- **信号**：连续3根阳线后回调到这3根的最低点 → 买入

# 📈 第三类：波动率因子 (15个)
**作用：风险管理+预判大行情**

## 39. GVZ (黄金VIX)
- **是什么**：CBOE黄金波动率指数，相当于黄金市场的"恐慌指数"
- **怎么用**：GVZ > 20 = 市场紧张 | > 30 = 恐慌 | < 15 = 平静
- **信号**：GVZ从15突然飙升到25 → 黄金当日涨2%+

## 40. VIX (美股VIX)
- **是什么**：美股恐慌指数
- **怎么用**：VIX涨 = 股市慌 = 黄金涨（避险）
- **信号**：VIX单日涨50% → 黄金涨3-5%

## 41. GVZ/VIX比率
- **是什么**：黄金恐慌 vs 股市恐慌
- **怎么用**：比率 > 1 = 黄金避险需求更强 | < 0.8 = 资金流向股市
- **信号**：比率从0.8升到1.2 → 黄金相对股市更强

## 42. VXO
- **是什么**：标普100波动率指数
- **怎么用**：类似VIX但针对蓝筹股
- **信号**：VXO飙升 → 黄金避险

---

## 43. 波动率期限价差 (3M-1M)
- **是什么**：3个月波动率 - 1个月波动率
- **怎么用**：远期波动率 > 近期 = Contango = 机构认为未来风险大
- **信号**：价差 > 2% → 机构看涨黄金波动率

## 44. 波动率期限斜率
- **是什么**：波动率期限结构的整体斜率
- **怎么用**：斜率为正 = 上涨趋势 = 机构看多
- **信号**：斜率从负转正 → 波动率要涨

## 45. 前端波动率
- **是什么**：1个月期权隐含波动率
- **怎么用**：反映近期市场风险
- **信号**：前端波动率突破20日高点 → 大行情要來

## 46. 远期波动率
- **是什么**：3-6个月期权隐含波动率
- **怎么用**：反映远期市场预期
- **信号**：远期 > 前端 → 市场看涨

---

## 47. IV-HV价差
- **是什么**：隐含波动率 - 历史波动率
- **怎么用**：IV > HV = 期权溢价高 = 市场在炒恐慌 | IV < HV = 恐慌过度
- **信号**：IV-HV > 5% → 黄金要涨（买恐慌后的反弹）

## 48. IV历史百分位
- **是什么**：当前IV在过去一年的位置
- **怎么用**：IV在90%分位 = 非常贵 | IV在10%分位 = 非常便宜
- **信号**：IV从90%分位回落 → 波动率要降

## 49. IV突破阈值
- **是什么**：IV突破20日均值+1.5个标准差
- **怎么用**：极端波动率信号
- **信号**：IV突破阈值 → 大行情预警

---

## 50. 波动率微笑
- **是什么**：虚值期权IV vs ATM期权IV的形态
- **怎么用**：左偏（put贵）= 市场怕跌 | 右偏（call贵）= 市场怕涨
- **信号**：左偏严重 → 黄金支撑强

## 51. Skew偏度
- **是什么**：波动率微笑的倾斜程度
- **怎么用**：Skew > 10 = 怕跌 | Skew < -10 = 怕涨
- **信号**：Skew从10降到-10 → 市场从怕跌变怕涨

## 52. 波动率聚类
- **是什么**：历史波动率往往一段时间高、一段时间低
- **怎么用**：低波动后必然高波动
- **信号**：波动率连续1个月 < 10 → 即将放大

---

## 53. FOMC事件
- **是什么**：美联储议息会议
- **怎么用**：会议前IV必然飙升，会议后IV必然回落
- **信号**：FOMC前3天 → 波动率买入机会

## 54. 非农事件
- **是什么**：美国就业数据发布
- **怎么用**：数据发布前后黄金波动剧烈
- **信号**：非农当天 → 波动率策略最佳

## 55. 地缘事件
- **是什么**：战争/制裁/危机
- **怎么用**：地缘爆发 → GVZ飙升 → 黄金涨
- **信号**：中东开打 → 黄金涨5-10%

# 📉 第四类：技术因子 (25个)
**作用：找到精确买卖点**

## 56-63. 均线系列
- **SMA20**：20日均线，站上=多头
- **SMA50**：50日均线，中期趋势
- **SMA200**：200日均线，牛熊分界线
- **EMA9**：9日指数均线，更灵敏
- **EMA50**：50日指数均线
- **EMA200**：200日指数均线
- **金叉**：短期均线>长期均线，买入信号
- **死叉**：短期均线<长期均线，卖出信号
- **怎么用**：站上200日均线 → 长期多头开始

## 64-73. 动量系列
- **RSI(14)**：相对强弱指标，>70超买要跌，<30超卖要涨
- **RSI(9)**：更敏感的RSI
- **MACD**：12日EMA-26日EMA，金叉买死叉卖
- **MACD柱状图**：MACD-Signal，柱子转负=顶背离
- **MACD背离**：价格新高但MACD没新高 = 顶背离要跌
- **Stochastic**：随机指标，%K%D交叉
- **Stochastic背离**：经典反转信号
- **CCI**：商品通道指标，>100超买，<-100超卖
- **ROC**：变动率
- **Momentum**：动量

## 74-77. 通道系列
- **布林带突破**：突破上轨强涨，跌破下轨强跌
- **布林带收窄**：收窄后必有突破（暴风雨前的宁静）
- **Keltner通道**：趋势通道
- **Donchian通道**：突破20日高点/低点

## 78-82. 成交量系列
- **成交量**：放量=真突破，缩量=假突破
- **成交量均线**：缩量=盘整
- **OBV**：能量潮，价格涨OBV不涨=背离
- **价量配合**：涨时放量=真涨，跌时缩量=假跌
- **价量背离**：价格涨但成交量缩=假突破

## 83-90. 形态系列
- **头肩顶**：经典顶部形态，三座山
- **头肩底**：经典底部形态，三个坑
- **双顶**：M头，两次上不去
- **双底**：W底，两次下不去
- **三角形突破**：收敛后爆发
- **旗形整理**：趋势中的中继
- **吞没形态**：大阳包大阴=强涨，大阴包阳=强跌
- **Pin Bar**：长影线反转

# 🀄 第五类：缠论因子 (10个)
**作用：最懂A股/期货的本土理论**

## 91. 笔
- **是什么**：最基本的走势单元，至少5根K线且高点、低点依次抬升/下降
- **怎么用**：笔是缠论最小单位
- **信号**：笔的结束 = 买卖点

## 92. 线段
- **是什么**：3笔构成线段，比笔更大级别
- **怎么用**：线段内笔的方向一致
- **信号**：线段破坏 = 大级别转折

## 93. 中枢
- **是什么**：连续3笔重叠区域 = 主力成本区间
- **怎么用**：中枢内高抛低吸
- **信号**：突破中枢后回抽不破 = 3买/3卖

## 94. 背驰/背离
- **是什么**：趋势力度衰减，比如价格新高但MACD没新高
- **怎么用**：背驰 = 必反转向
- **信号**：底背离 = 买入 | 顶背离 = 卖出

## 95. 第一类买卖点
- **是什么**：背驰发生的点，最佳买卖点
- **怎么用**：这是缠论最看重的点
- **信号**：出现一买/一卖

## 96. 第二类买卖点
- **是什么**：中枢形成后，笔离开中枢又回来，不破中枢边界
- **怎么用**：一买后第一次回抽
- **信号**：二买 = 确认买入
