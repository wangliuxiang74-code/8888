## 第四章：情绪评分

### 4.1 评分方法
```
每条新闻打分:
利好词 +1分
利空词 -1分
紧急词 ×2

总分:
> 3 = 极度看涨 🟢🟢
1-3 = 轻微看涨 🟢
-1~1 = 中性 🟡
-3~-1 = 轻微看跌 🔴
< -3 = 极度看跌 🔴🔴
```

### 4.2 加权计算
```
最近1小时新闻: 权重 50%
最近3小时新闻: 权重 30%
最近24小时新闻: 权重 20%

最终分数 = 50%×1h + 30%×3h + 20%×24h
```

## 第五章：实战流程

### 5.1 每日流程
```
1. 开盘前 (9:00)
   - 浏览隔夜重大新闻
   - 给出当日基调

2. 盘中 (每小时)
   - 监控实时新闻
   - 及时预警

3. 收盘后 (17:00)
   - 总结今日新闻
   - 给出明日展望
```

### 5.2 实时监控
```python
# 新闻监控伪代码
def monitor_news():
    while market_open:
        news = fetch_latest_news()
        
        # 提取关键词
        keywords = extract_keywords(news)
        
        # 情绪打分
        score = calculate_sentiment(keywords)
        
        # 更新信号
        if score > threshold:
            send_alert("BULLISH", score)
        elif score < -threshold:
            send_alert("BEARISH", score)
        
        # 等待下一条
        wait(60 seconds)
```
