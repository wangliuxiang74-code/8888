## 第二章：数据发布解读

### 2.1 非农数据
```
前值: 上月数据
预期: 市场预测
实际: 公布值

解读:
- 实际 > 预期 + 利好美元 → 黄金跌
- 实际 < 预期 - 利空美元 → 黄金涨
- 实际 = 预期 → 震荡
```

### 2.2 CPI数据
```
核心CPI: 剔除食品和能源
影响:
- CPI高 → 加息预期 → 黄金跌
- CPI低 → 降息预期 → 黄金涨
```

### 2.3 GDP数据
```
- GDP高 → 经济好 → 黄金跌
- GDP低 → 经济差 → 黄金涨
```

## 第五章：持仓数据

### 5.1 ETF持仓
```python
# GLD持仓变化
today_holdings = 900  # 吨
yesterday_holdings = 880
change = today - yesterday = +20吨

# 解读
change > 10吨 → 机构买入
change < -10吨 → 机构卖出
```

### 5.2 COT持仓
```python
# 大型投机商
spec_long = 250000  # 手
spec_short = 100000
net = spec_long - spec_short = +150000

# 解读
net > 200000 → 超买 → 黄金可能跌
net < -100000 → 超卖 → 黄金可能涨
```
