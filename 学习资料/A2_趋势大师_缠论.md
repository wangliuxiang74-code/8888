## 第一章：缠论基础

### 1.1 缠论核心概念
```
笔 → 线段 → 中枢 → 背驰 → 买卖点
```

### 1.2 笔
- **定义**：至少5根连续K线，走势向上必须有新高+新低
- **方向**：向上笔=创新高+创新低；向下笔=新低+新低
- **简化**：看K线形态

### 1.3 线段
- **定义**：3笔构成线段
- **方向**：线段方向与笔一致
- **破坏**：被反向笔突破

### 1.4 中枢
- **定义**：连续3笔重叠区域
- **作用**：主力成本区间，支撑/阻力
- **级别**：笔中枢、线段中枢

## 第二章：买卖点

### 2.1 第一类买卖点
- **位置**：趋势背驰点
- **信号**：价格创新高/低，MACD不创新高/低
- **特点**：最高/最低点，风险大

### 2.2 第二类买卖点
- **位置**：中枢完成后，笔回到中枢边界
- **信号**：回抽不破中枢
- **特点**：确认信号，相对安全

### 2.3 第三类买卖点
- **位置**：突破中枢后，笔回到中枢但不进入
- **信号**：突破后的回抽确认
- **特点**：最强信号

## 第三章：背驰/背离

### 3.1 背驰类型

| 类型 | 条件 | 信号 |
|------|------|------|
| 价背 | 价格新高，MACD柱状图新低 | 顶背驰=卖 |
| 量背 | 价格新高，成交量不放大 | 顶背驰=卖 |
| 价底 | 价格新低，MACD柱状图新高 | 底背驰=买 |
| 量底 | 价格新低，成交量萎缩 | 底背驰=买 |

### 3.2 判断方法
```
1. 看MACD柱子：红柱缩短=多头力度弱
2. 看均线：价格涨但均线斜率变缓
3. 看RSI：创新高/低但RSI背离
4. 看成交量：价涨量缩
```

## 第四章：中枢与笔

### 4.1 中枢画法
```
1. 找到连续3笔重叠的部分
2. 重叠区间就是中枢
3. 中枢区间越高，压力越大
4. 中枢区间越低，支撑越强
```

### 4.2 中枢操作
```
- 在中枢内：高抛低吸
- 突破中枢：顺势交易
- 中枢破坏：趋势反转
```

## 第五章：技术指标配合

### 5.1 MACD
- **参数**：12,26,9
- **用法**：
  - 金叉=买入，死叉=卖出
  - 0轴上方强，下方弱
  - 背离=最佳买卖点

### 5.2 RSI
- **参数**：14
- **用法**：
  - >70 超买，可能跌
  - <30 超卖，可能涨
  - 背离=反转信号

### 5.3 布林带
- **参数**：20,2
- **用法**：
  - 突破上轨=强涨
  - 突破下轨=强跌
  - 收窄=突破在即

## 第六章：M2周期实战

### 6.1 日线/4小时线
- **周期**：1小时-4小时
- **持仓**：1-3天
- **目标**：100-300点

### 6.2 交易流程
```
1. 日线定方向
2. 4小时找中枢
3. 1小时找买卖点
4. 30分钟精确入场
```

### 6.3 入场条件
```
A. 趋势确认：
   - 日线站上20日均线
   
B. 中枢形成：
   - 4小时形成中枢
   
C. 买卖点：
   - 出现底背离
   - 30分钟突破
   
D. 确认：
   - 成交量放大
   - RSI配合
```

## 第七章：平仓信号

### 7.1 止盈
```
- 目标1：100点 → 出一半
- 目标2：200点 → 再出一半
- 目标3：300点 → 清仓
```

### 7.2 止损
```
- 固定2%
- 跌破前低
- 出现相反背离
- 突破中枢下沿
```

### 7.3 时间止损
```
- 持仓超过24小时无动静 → 平仓
- 持仓超过3天 → 强制平仓
```
