## 第一章：波浪理论

### 1.1 艾略特波浪
```
推动浪: 1-2-3-4-5
调整浪: A-B-C

5浪上涨 = 3浪下跌
```

### 1.2 黄金波浪
```
1浪: 启动 (4900→4950)
2浪: 回撤 (4950→4920)
3浪: 主升 (4920→5100) 最强
4浪: 整理 (5100→5050)
5浪: 冲刺 (5050→5200)
```

### 1.3 浪型识别
```
- 3浪不能最短
- 4浪不能跌破1浪顶
- 2浪回撤不能超过100%
```

## 第五章：综合实战

### 5.1 多周期分析
```
日线: 上升趋势 (站上20日均线)
4小时: 3浪上涨
1小时: 回调到38.2%
30分钟: 出现底背离

结论: 3浪回调结束，准备做多
```

### 5.2 缠论+波浪
```
- 1浪 = 向上笔
- 2浪 = 笔回撤
- 3浪 = 线段上涨
- 4浪 = 笔回调
- 5浪 = 线段延伸
```

### 5.3 斐波那契+中枢
```
- 回调到中枢上沿
- 同时是38.2%回调位
- 出现底背离

= 最佳买点
```
