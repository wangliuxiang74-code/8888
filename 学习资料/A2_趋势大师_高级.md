## 第一章：笔的精确定义

### 1.1 严格笔
```
向上笔: 连续5根K线,高点依次抬高,低点依次抬高
向下笔: 连续5根K线,高点依次降低,低点依次降低

缺口处理:
- 向上跳空不破坏笔
- 向下跳空破坏笔
```

### 1.2 笔的破坏
```
新笔没有破坏旧笔:
- 向上笔中,后续K线低点 > 前面笔的低点

新笔破坏旧笔:
- 向上笔中,后续K线低点 < 前面笔的低点
- 向下笔中,后续K线高点 > 前面笔的高点
```

## 第二章：线段

### 2.1 线段定义
```
线段: 由3笔构成,方向相同

向上线段:
- 第1笔起点是线段起点
- 第3笔终点是线段终点
- 中间笔的高点/低点依次抬升

向下线段相反
```

### 2.2 线段破坏
```
特征:
- 反向一笔突破线段起点
- 破坏K线>=3根

意义:
- 线段破坏 = 大级别转折
- 可能形成新的线段
```

## 第三章：中枢

### 3.1 中枢画法
```
三笔重叠区间:
笔1: 4900→4950
笔2: 4930→4980 (与笔1重叠: 4930-4950)
笔3: 4960→5010 (与笔2重叠: 4960-4980)

中枢区间: [4960, 4980]
```

### 3.2 中枢级别
```
笔中枢: 1小时以下
线段中枢: 4小时-日线
笔线段中枢: 日线-周线
```

### 3.3 中枢操作
```
在中枢内:
- 上沿卖出,下沿买入

突破中枢:
- 向上突破: 回调不破中枢上沿 = 3买
- 向下突破: 反弹不破中枢下沿 = 3卖
```

## 第四章：背驰

### 4.1 力度比较
```
比较方法:
1. MACD柱子面积
2. 均线斜率
3. K线数量
4. 幅度

背驰判断:
- 涨幅变小但创新高 = 顶背驰
- 跌幅变小但创新低 = 底背驰
```

### 4.2 背驰级别
```
本级别背驰:
- 导致笔的反转

大级别背驰:
- 导致线段/中枢的反转
- 可能是日线/周线级别
```

## 第五章：三类买卖点

### 5.1 第一类买卖点
```
位置:
- 趋势最后一个中枢后
- 背驰点

特征:
- 最高点/最低点
- 确定性低,需要次级别确认

确认方法:
- 出现反方向笔
- 突破中枢
```

### 5.2 第二类买卖点
```
位置:
- 第一类买卖点后
- 中枢区间内

特征:
- 不创新高/低
- 确定性中等

确认方法:
- 回到中枢内
```

### 5.3 第三类买卖点
```
位置:
- 中枢外
- 突破后回抽

特征:
- 离开中枢后不回到中枢
- 确定性高

信号:
- 3买: 突破后回调不破中枢上沿
- 3卖: 突破后反弹不破中枢下沿
```

## 第六章：量化实现

### 6.1 笔识别
```python
def find_bi(df):
    """识别笔"""
    direction = None
    bi_start = None
    bi_list = []
    
    for i in range(len(df)):
        if direction is None:
            direction = 'up' if df['close'][i] > df['open'][i] else 'down'
            bi_start = i
        else:
            # 检查方向
            if direction == 'up':
                if df['low'][i] < df['low'][bi_start]:  # 破坏
                    # 形成笔
                    bi_list.append({
                        'start': bi_start,
                        'end': i-1,
                        'type': 'up'
                    })
                    direction = 'down'
                    bi_start = i
            else:
                if df['high'][i] > df['high'][bi_start]:  # 破坏
                    bi_list.append({
                        'start': bi_start,
                        'end': i-1,
                        'type': 'down'
                    })
                    direction = 'up'
                    bi_start = i
    
    return bi_list
```

### 6.2 中枢识别
```python
def find_zhongshu(bi_list):
    """识别中枢"""
    zhongshu_list = []
    
    for i in range(len(bi_list) - 2):
        bi1 = bi_list[i]
        bi2 = bi_list[i+1]
        bi3 = bi_list[i+2]
        
        # 检查是否有重叠
        if bi1['type'] == bi3['type']:
            high = min(bi1['high'], bi2['high'], bi3['high'])
            low = max(bi1['low'], bi2['low'], bi3['low'])
            
            if high > low:  # 有重叠
                zhongshu_list.append({
                    'high': high,
                    'low': low,
                    'mid': (high + low) / 2
                })
    
    return zhongshu_list
```

                        })
