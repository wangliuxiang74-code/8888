## 第一章：剥头皮战法 (Scalping)

### 1.1 什么是剥头皮
- 时间周期：1分钟/5分钟
- 目标：5-20点利润
- 持仓：几秒到几分钟
- 胜率要求：>60%

### 1.2 剥头皮核心原则
1. **快进快出** - 不持仓过夜
2. **见利就跑** - 5点就跑
3. **错了就砍** - 3点必须止损
4. **轻仓** - 每次0.1手
5. **顺势** - 只做顺势单

### 1.3 剥头皮时间
| 时段 | 波动 | 适合 |
|------|------|------|
| 13:00-17:00 UTC | 大 | 最佳 |
| 21:00-00:00 UTC | 大 | 最佳 |
| 00:00-07:00 UTC | 小 | 不做 |
| 数据发布时 | 极大 | 不做 |

### 1.4 剥头皮信号
1. **突破**：突破前高/前低
2. **回撤**：回调到均线支撑
3. **背离**：1分钟MACD背离
4. **整理突破**：三角形/旗形突破

## 第二章：对冲挂单 (Grid Trading)

### 2.1 原理
在震荡区间上下沿同时挂买单和卖单，区间内每单都对冲平仓。

### 2.2 网格设置
```
例如：黄金在4900-5100震荡
4900: 多单挂单
4920: 多单挂单
...
5080: 空单挂单
5100: 空单挂单
每格间距: 20点
```

### 2.3 挂单技巧
1. **关键位挂单**：支撑/阻力位
2. **突破挂单**：挂突破单
3. **反弹挂单**：挂回踩单
4. **间距**：20-50点

### 2.4 风控
- 总额度：不超过20%仓位
- 单边：不超过3单
- 止损：总资金2%

## 第三章：订单流 (Order Flow)

### 3.1 核心概念

#### Delta (δ)
- 主动性买单成交量 - 主动性卖单成交量
- Delta > 0：多头主动
- Delta < 0：空头主动

#### Cumulative Delta (累计Delta)
- 一天内Delta累计
- 持续为正：多头控盘
- 持续为空：空头控盘

#### TPO (Time Price Opportunity)
- 价格在每个价位停留的时间
- 停留长 = 供给/需求区
- 停留短 = 突破

### 3.2 订单流信号

| 信号 | 说明 | 交易 |
|------|------|------|
| Delta持续为正 | 多头控盘 | 做多 |
| Delta持续为负 | 空头控盘 | 做空 |
| 价格新高但Delta不新高 | 顶背离 | 做空 |
| 价格新低但Delta不新低 | 底背离 | 做多 |
| TPO密集区 | 支撑/阻力 | 挂单 |

### 3.3 成交量分布
- **高成交量区域**：往往成为支撑/阻力
- **低成交量区域**：容易被突破
- **成交量放大**：方向确认

## 第四章：做市商套利 (Market Making)

### 4.1 原理
做市商提供流动性，低价买高价卖，赚取价差。

### 4.2 策略

#### 被动挂单
- 在bid挂买单
- 在ask挂卖单
- 等待成交

#### 快速平仓
- 成交后立即平仓
- 目标：1-3点利润
- 止损：5点

### 4.3 技巧
1. **流动性好的时段**：伦敦/纽约
2. **波动小的时段**：亚洲早盘
3. **价差小的平台**：ECN账户
4. **高杠杆**：100倍+

### 4.4 风险
- 方向做反
- 流动性枯竭
- 滑点

## 第五章：1分钟战法

### 5.1 均线用法
- EMA9：短期趋势
- EMA21：中期趋势
- 价格站上EMA9+EMA21 → 多
- 价格跌破EMA9+EMA21 → 空

### 5.2 MACD用法
- 0轴上方：金叉做多
- 0轴下方：死叉做空
- 背离：1分钟顶底背离

### 5.3 布林带用法
- 突破上轨：做多
- 突破下轨：做空
- 收窄：等待突破

### 5.4 黄金1分钟特殊
- 波动大：止损要宽
- 点差大：做够cover成本
- 流动性：亚洲盘少做

## 第六章：实战案例

### 案例1：突破剥头皮
```
时间：14:30 UTC
形态：1分钟三角形突破
信号：价格突破4900，成交量放大
入场：4901做多
止损：4898
止盈：4910
结果：+9点
```

### 案例2：对冲挂单
```
区间：4880-4920
挂单：4880多，4900多，4920空
成交：4900多单成交
平仓：4910平多，-10点
净收益：0（对冲）
```

### 案例3：订单流
```
信号：Cumulative Delta持续为正
价格：4900
入场：4901做多
止损：4895
止盈：4920
结果：+19点
```
