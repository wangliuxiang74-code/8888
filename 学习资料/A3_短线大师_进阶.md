## 第一章：深度订单流

### 1.1 Volume Profile
```
- TPO (Time Price Opportunity)
- POC (Point of Control) - 成交量最大价位
- VA (Value Area) - 70%成交量区间
- VAL (Value Area Low)
- VAH (Value Area High)
```

### 1.2 应用
```
- 价格在POC上方 → 多头控盘
- 价格在POC下方 → 空头控盘
- 价格突破POC → 趋势延续
```

### 1.3 Delta分析
```
- Delta = 主动性买入 - 主动性卖出
- 持续正值 → 多头主动
- 持续负值 → 空头主动
- Delta反转 → 趋势反转
```

## 第二章：流动性分析

### 2.1 流动性池
```
- 大额限价单 (Iceberg)
- 暗池流动性
- Stop Loss 集群
```

### 2.2 流动性获取
```
- 突破关键价位
- 扫掉止损
- 流动性耗尽
```

### 2.3 订单簿分析
```
Bid Size: 卖单深度
Ask Size: 买单深度
Ratio: 深度比

- Bid > Ask × 2 → 多头强势
- Ask > Bid × 2 → 空头强势
```

## 第三章：市场微观结构

### 3.1 市场深度
```
- Level 1: 最佳买卖价
- Level 2: 5档深度
- Level 3: 10档深度
```

### 3.2 价差分析
```
- 正常价差: 10-20点
- 紧张价差: 5-10点
- 极窄: <5点 → 可能突破
```

### 3.3 成交分布
```
- 成交集中在POC附近
- POC是支撑/阻力
- POC变化 = 方向变化
```

## 第四章：高频策略

### 4.1 突破策略
```
条件:
- 价格突破VAH
- 成交量放大
- Delta为正

入场: VAH + 5点
止损: POC下方
目标: 2R
```

### 4.2 回调策略
```
条件:
- 价格回到VAL
- POC未破
- 出现吸筹形态

入场: VAL + 5点
止损: VAL下方
目标: POC
```

### 4.3 假突破策略
```
条件:
- 突破后快速回落
- Delta转负
- 成交量萎缩

入场: 突破点下方10点
止损: 突破高点
目标: 回归POC
```

## 第五章：算法交易

### 5.1 冰山订单
```
Iceberg Order:
- 显示1手，实际100手
- 被动成交策略
- 等待被动买入
```

### 5.2 TWAP
```
Time Weighted Average Price:
- 分时成交
- 降低冲击成本
- 适合大单
```

### 5.3 执行算法
```
- VWAP: 成交量加权
- POV: Percentage of Volume
- IS: Implementation Shortfall
```
