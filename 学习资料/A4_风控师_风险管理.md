## 第一章：风控核心原则

### 1.1 核心原则
1. **活着最重要** - 不亏钱 > 赚钱
2. **宁可错过** - 不做不错
3. **错了就砍** - 止损要快
4. **仓位要轻** - 留得青山在
5. **日亏2%** - 当日必须停

### 1.2 风控铁律
```
1. 单笔亏损 > 2% = 必须平仓
2. 日亏损 > 3% = 停止交易
3. 连续3笔亏损 = 停止交易
4. 点差 > 30点 = 不做
5. 波动率 > 30 = 不做
6. 新闻前30分钟 = 不做
7. 持仓过夜 = 尽量避免
```

## 第二章：数值敏感

### 2.1 必须记住的数字
| 指标 | 安全值 | 警告值 | 危险值 |
|------|--------|--------|--------|
| 点差 | < 20 | 20-30 | > 30 |
| 波动率 | < 15 | 15-30 | > 30 |
| 持仓数 | < 2 | 2-3 | > 3 |
| 日亏损 | < 1% | 1-3% | > 3% |
| 仓位 | < 10% | 10-20% | > 20% |

### 2.2 风险计算
```python
# 仓位计算公式
仓位 = 账户余额 × 风险% ÷ 止损点数

# 例如：账户10万，风险1%，止损20点
仓位 = 100000 × 0.01 ÷ 20 = 50手

# 实际仓位不能超过最大仓位
最大仓位 = 账户余额 × 20% ÷ 当前价格
```

### 2.3 盈亏比
```
最小盈亏比 = 1.5 : 1

计算：
止损 = 20点
止盈 = 30点 (1.5倍)
胜率 = 40%
数学期望 = 40% × 30 - 60% × 20 = 0

要求：
胜率 > 40% 才能盈利
```

## 第四章：风控策略

### 4.1 固定比例风控
```python
# 每次交易风险固定比例
每次风险% = 1%

# 账户10万
单笔最大亏损 = 1000元

# 止损20点
每点价值 = 1000 ÷ 20 = 50元
仓位 = 50元 ÷ 黄金价格5000 = 0.01手
```

### 4.2 波动率风控
```python
# 根据波动率调整仓位
ATR = 计算20日ATR
止损点数 = ATR × 2

# ATR = 50点
止损 = 100点
仓位 = 账户×1% ÷ 100点
```

### 4.3 趋势风控
```python
# 趋势中用移动止损
入场后：
  +1% → 移动止损到成本
  +2% → 移动止损到+1%
  +3% → 移动止损到+2%
```

### 4.4 时间风控
```python
# 持仓时间限制
M2周期：持仓 < 24小时
M1周期：持仓 < 1小时
剥头皮：持仓 < 15分钟

# 超过时间强制平仓
if 持仓时间 > 最大时间:
    强制平仓()
```

## 第五章：紧急情况

### 5.1 熔断规则
```
日亏损 > 3% = 熔断停止交易
连续3笔亏损 = 熔断停止交易
波动率 > 40 = 停止开仓
```

### 5.2 黑天鹅应对
```
1. 立即平掉所有仓位
2. 不要试图抄底
3. 等待稳定后再入场
4. 记录事件分析原因
```

### 5.3 流动性枯竭
```
1. 挂单价差扩大
2. 无法成交
3. 成交价格滑落大
4. 应对：市价成交/撤单等待
```
