## 第一章：因子融合

### 1.1 输入源
| 分析师 | 输入 | 权重 |
|--------|------|------|
| A1 | 新闻情绪 | 25% |
| A2 | 缠论信号 | 35% |
| A3 | 短线信号 | 20% |
| A4 | 风控审核 | 一票否决 |

### 1.2 融合公式
```
综合分数 = A1×0.25 + A2×0.35 + A3×0.20

决策:
分数 ≥ 0.6 → 做多
分数 ≤ -0.6 → 做空
其他 → 观望
```

### 1.3 决策流程
```
A1分析 → A2分析 → A3分析 → 融合分数
    ↓
提交A4审核
    ↓
通过 → 执行
否决 → 重新论证
```

## 第二章：仓位管理

### 2.1 基础仓位
```
账户10万，单笔风险1% = 1000元
止损20点 = 每点50元
仓位 = 0.01手
```

### 2.2 仓位公式
```
仓位 = 账户余额 × 风险% ÷ 止损点数
```

### 2.3 加减仓
```
- 趋势确认: +20%
- 背离信号: -20%
- 波动率高: -30%
- 连续盈利: +10%
```

## 第三章：策略进化

### 3.1 每日复盘
```
记录:
- 信号准确率
- 盈利/亏损
- 仓位是否合适
- 风控是否及时
```

### 3.2 周度优化
```
分析:
- 哪些信号胜率高
- 哪些信号需要调整
- 因子权重是否需要变
```

### 3.3 月度进化
```
评估:
- 月收益率
- 夏普比率
- 最大回撤
- 策略是否需要大改
```

## 第四章：决策系统

### 4.1 信号强度
| 分数 | 信号 | 仓位 |
|------|------|------|
| > 0.8 | 强买 | 2% |
| 0.6-0.8 | 买入 | 1.5% |
| 0.4-0.6 | 轻买 | 1% |
| -0.4-0.4 | 观望 | 0% |
| -0.6~-0.4 | 轻卖 | 1% |
| -0.8~-0.6 | 卖出 | 1.5% |
| < -0.8 | 强卖 | 2% |

### 4.2 风控优先
```
任何时候:
- A4否决 = 终止
- 风险超标 = 减仓
- 连续亏损 = 停止
```

## 第五章：实战案例

### 案例：多分析师融合
```
A1: 新闻情绪 +0.5 (利好)
A2: 缠论信号 +0.7 (二买)
A3: 短线信号 +0.3 (震荡)

融合分数: 0.5×0.25 + 0.7×0.35 + 0.3×0.20 = 0.505

提交A4审核:
- 点差: 15 ✓
- IV: 20 ✓
- 持仓: 1单 ✓

结果: 通过 → 买入1%
```
